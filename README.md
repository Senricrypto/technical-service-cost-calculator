# Technical Service Cost Calculator

This project calculates the cost of a technical service operation based on distance and average speed.

## 🚀 Features
- Calculates service cost based on distance
- Estimates travel time
- Simple and user-friendly input system

## 🛠 Technologies Used
- Python
- Jupyter Notebook

## 📊 Example Usage

User inputs:
- Distance (km)
- Average speed (km/h)

Output:
- Estimated travel time
- Total service cost

## 🎯 Purpose
This project was created as a beginner-level Python project to practice:
- User input handling
- Basic calculations
- Code optimization

## 🔄 Improvements
- Simplified input conversion (int(input()))
- Reduced unnecessary variables
- Improved code readability

## 📌 Future Improvements
- Add error handling (try-except)
